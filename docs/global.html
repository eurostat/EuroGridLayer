viewer()

Generates a 2D Three.js scene for visualizing large point datasets using WebGL. The library follows a similar structure to that of d3, whereby parameters are set using a series of accessor functions, each of which returns the main viewer.
Parameters:
Name Type Attributes Default Description
viewer.container_element() HTMLElement <optional>
 document.body The DOM element used for the THREE.js renderer.
viewer.height number <optional>
 window.innerHeight The height value used for the threejs scene, camera, zoom, labels.
viewer.width number <optional>
 window.innerWidth The width value used for the threejs scene, camera, zoom, labels.
viewer.backgroundColor number <optional>
 "#b7b7b7" THREE.Scene background color (hexidecimal).
viewer.borderColor number <optional>
 "#ffffff" Hex string which is later converted to THREE.Line2 color (hexidecimal).
viewer.highlightColor string <optional>
 "#37f2d6" Hex string which is later converted to THREE.Line2 color (hexidecimal).
viewer.loadingIcon string <optional>
 "ring" CSS animation used for the loading icon, other options are: ripple | ring | ellipsis | roller.
viewer.colorSchemeName string <optional>
 "interpolateTurbo" Name of the d3-sclae-chromatic function name used for colouring cells.
viewer.colors array <optional>
 null Array of hex strings to be used in combination with viewer.thresholdValues for colouring.
viewer.thresholdValues array <optional>
 null Array of threshold values that are applied to the colour scaling function.
viewer.colorScaleFunction function <optional>
 d3.scaleSequential D3.scale function used for defining colour values. Current working functions seem to be scaleSequential or scaleDiverging & their respective variants.
viewer.sizeScaleFunction function <optional>
 d3.scaleSqrt D3.scale function used for defining colour values. Current working functions seem to be scaleSequential or scaleDiverging & their respective variants.
viewer.colorSchemeSelector boolean <optional>
 false Whether or not to build a simple html select input for the available d3-scale-chromatic colour schemes.
viewer.colorFieldSelector boolean <optional>
 false Whether or not to build a simple html select input showing the available fields in the csv file that can be used to drive cell colour.
viewer.sizeFieldSelector boolean <optional>
 false Whether or not to build a simple html select input showing the available fields in the csv file that can be used to drive cell size.
viewer.showLegend boolean <optional>
 true Build d3 colour legend.
viewer.showPlacenames boolean <optional>
 true Adds placenames to the viewer. Placename-scale thresholds currently defined by the user.
viewer.showColorSchemeSelector boolean <optional>
 true THREE.Line2 color (hexidecimal).
viewer.legend Object <optional>
 Builds d3 colour legend.
